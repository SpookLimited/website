Access controls

The areas that OmniWatch traditionally monitor often spread across several departmental remits which can be difficult to manage or keep secure. For example a refrigeration alarm would need to inform both the maintenance/service company and those that use the equipment on site. Granting and managing access to multiple users in such a case would be time consuming and potentially expose users to information they either don't need or shouldn't see.

OmniWatch solves this issue and streamlines the workflow for the end user with it's built in access controls. The controls are flexible and can not only be used for sensors and metrics on a single device but also allows for filtering by location, equipment type or however the user wishes.

Custom sensor settings

Spook recognises similar sensors may need different settings dependant where they are installed. For example a temperature sensor near an air conditioning unit may need lower variables when compared to a similar sensor located near to a heat source.

All sensor settings are variable and can be easily changed via the individual sensor interface within OmniWatch.
